# Components

This folder will contain reusable UI components built in Phase 2:

- `KpiGrid.tsx` — top KPI cards (price, market cap, volume, etc.)
- `VolumeChart.tsx` — daily trading volume bar chart
- `OrderBookChart.tsx` — bid/ask depth over time
- `LiquidityChart.tsx` — stacked bar by exchange over time
- `RadarChart.tsx` — peer efficiency radar
- `ConversionSection.tsx` — retainer offer + email gate
